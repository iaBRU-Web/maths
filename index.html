Question 1
Evaluate lim(x→−4) of (3 − √(25 − x²)) / (x + 4)

Direct substitution gives 0/0 (indeterminate). Rationalize by multiplying numerator and denominator by the conjugate:

[3 − √(25−x²)] · [3 + √(25−x²)]
——————————————————————————
(x+4) · [3 + √(25−x²)]

= (9 − (25−x²)) / [(x+4)(3+√(25−x²))]
= (x²−16) / [(x+4)(3+√(25−x²))]
= (x+4)(x−4) / [(x+4)(3+√(25−x²))]
= (x−4) / [3 + √(25−x²)]

Now substitute x = −4:

= (−4−4) / [3 + √(25−16)] = −8 / [3+3] = −8/6
∴ The limit = −4/3
Question 2
The lines x=1, y+5=0 and x−2=0 are asymptotes of f(x) = cx² / (ax²−3x−b). Find a, b and c.

Vertical asymptotes at x=1 and x=2 mean the denominator factors as a(x−1)(x−2):

a(x−1)(x−2) = a(x²−3x+2) = ax²−3ax+2a

Comparing with ax²−3x−b: coefficient of x gives −3a = −3 → a = 1; constant term: 2a = −b → b = −2.

Horizontal asymptote y = −5: as x→∞, f(x) → c/a = c/1 = c = −5.

a = 1, b = −2, c = −5
Question 3
Show that f(x) = (2x−3)/(x+1) and g(x) = (x+3)/(2−x) are inverse to each other.

Compute f(g(x)):

f(g(x)) = [2·(x+3)/(2−x) − 3] / [(x+3)/(2−x) + 1]
Numerator: [2(x+3) − 3(2−x)] / (2−x) = [2x+6−6+3x]/(2−x) = 5x/(2−x)
Denominator: [(x+3) + (2−x)]/(2−x) = 5/(2−x)
∴ f(g(x)) = 5x/5 = x ✓

Compute g(f(x)):

g(f(x)) = [(2x−3)/(x+1) + 3] / [2 − (2x−3)/(x+1)]
Numerator: [2x−3+3(x+1)]/(x+1) = [5x]/(x+1)
Denominator: [2(x+1)−(2x−3)]/(x+1) = 5/(x+1)
∴ g(f(x)) = 5x/5 = x ✓
Since f(g(x)) = g(f(x)) = x, the functions are inverses. ∎

Question 7
α and β are roots of 10x² − x + 1 = 0. Find α³ + β³.

By Vieta's formulas: α + β = 1/10, αβ = 1/10

α² + β² = (α+β)² − 2αβ = 1/100 − 2/10 = 1/100 − 20/100 = −19/100

α³ + β³ = (α+β)(α²−αβ+β²) = (α+β)[(α²+β²) − αβ]
= (1/10)[−19/100 − 1/10]
= (1/10)[−19/100 − 10/100]
= (1/10)(−29/100)
α³ + β³ = −29/1000
Question 8

Question 14
Given f(x) = √(x²+x+1) + x + 2. Find asymptotes.
a) Horizontal asymptote as x → −∞:
As x→−∞: √(x²+x+1) = |x|√(1+1/x+1/x²) = −x(1 + 1/(2x) + …) ≈ −x − 1/2
f(x) ≈ (−x − 1/2) + x + 2 = 3/2
y = 3/2 is the horizontal asymptote as x → −∞
b) Oblique asymptote as x → +∞:
As x→+∞: √(x²+x+1) ≈ x + 1/2
f(x) ≈ (x + 1/2) + x + 2 = 2x + 5/2
y = 2x + 5/2 is the oblique asymptote as x → +∞
Question 15
Find the perimeter, area and angles of triangle with vertices A(1,−5), B(4,−2) and C(−3,0).
Triangle ABC plotted on coordinate grid

AB = √[(4−1)²+(−2+5)²] = √(9+9) = 3√2 ≈ 4.243
AC = √[(−3−1)²+(0+5)²] = √(16+25) = √41 ≈ 6.403
BC = √[(−3−4)²+(0+2)²] = √(49+4) = √53 ≈ 7.280
Perimeter = 3√2 + √41 + √53 ≈ 17.93 units

Area (coordinate formula):

Area = ½|x_A(y_B−y_C) + x_B(y_C−y_A) + x_C(y_A−y_B)|
= ½|1(−2−0) + 4(0+5) + (−3)(−5+2)|
= ½|−2 + 20 + 9| = ½ · 27
Area = 27/2 = 13.5 sq. units

Angles (cosine rule):
